Array Methods

назад

Основные методы — что делают, примеры и советы

Mutating (мутирующие)

  
    arr.push(item)    // добавляет в конец, возвращает новую length
    arr.pop()         // удаляет последний, возвращает элемент
    arr.shift()       // удаляет первый (O(n)), возвращает элемент
    arr.unshift(x)    // вставляет в начало (O(n)), возвращает length
    arr.splice(i, n, ...items) // удаляет/вставляет; возвращает удалённые элементы
    arr.sort(fn)      // сортирует на месте, возвращает тот же массив
    arr.reverse()     // на месте
    arr.fill(v, start?, end?) // заполняет на месте

Non-mutating (не мутируют)

  
    const mapped = arr.map(x => x * 2)        // преобразование
    const filtered = arr.filter(x => x.active) // отбор
    const sliced = arr.slice(0, 10)           // копия части
    const concat = arr.concat(other)          // новый массив
    const flat = arr.flat(depth)              // группировка вложенных массивов
    const flatMap = arr.flatMap(x => [x, x*2])// map + flatten на 1 уровень
